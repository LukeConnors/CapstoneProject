## About The Project

[![Pledge Palooza Screenshot][product-screenshot]](https://trivia-titan.onrender.com/)

<p align="center">
Trivia Titan is a trvia game application that has exciting features and utilizes impressive technologies. Its front-end leverages React, Redux, and CSS to deliver a seamless user experience. On the backend, It utilizes Flask, SQLAlchemy, and SQLite to ensure efficiency and that the project remains lightweight.
</p>

<p align="center">
Authenticated users may do many things on the application, from creating their own trivia questions and decks to leaving reviews on decks and direct messaging other users using a websocket direct messaging system. Managing your decks and questions is simple through the "My Profile Page", and the game itself is amusing! The game page automatically shuffles answers to questions each time and tracks stats as the user answers questions.
</p>

<p align="center">
The user profile pages host lifetime stats of each user and even has a badging system where users can unlock unique badges as they complete trivia questions. This all harbors potential for community building. As does the addition of reviews and direct messaging!
</p>

## Getting started
1. Clone this repository (only this branch)

2. Install dependencies

      ```bash
      pipenv install -r requirements.txt
      ```

3. Create a **.env** file based on the example with proper settings for your
   development environment

4. Make sure the SQLite3 database connection URL is in the **.env** file

5. This starter organizes all tables inside the `flask_schema` schema, defined
   by the `SCHEMA` environment variable.  Replace the value for
   `SCHEMA` with a unique name, **making sure you use the snake_case
   convention**.

6. Get into your pipenv, migrate your database, seed your database, and run your Flask app

   ```bash
   pipenv shell
   ```

   ```bash
   flask db upgrade
   ```

   ```bash
   flask seed all
   ```

   ```bash
   flask run
   ```

7. To run the React App in development, checkout the [README](./react-app/README.md) inside the `react-app` directory.
